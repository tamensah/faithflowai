# FaithFlow AI Brand Guide

## Brand Essence
- Positioning: trustworthy, modern, and operationally precise
- Promise: reduce admin burden while preserving integrity and clarity
- Personality: calm authority, helpful, never flashy

## Voice and Tone
- Clear, concise, respectful
- Avoid hype and exaggerated claims
- Use plain language in user-facing copy

## Visual System
### Color Palette
- Primary: Deep Navy `#163A5F`
- Secondary: Teal `#2C7A78`
- Accent: Warm Sand `#E3B778`
- Background: Ivory `#F7F5F0`
- Surface: White `#FFFFFF`
- Text: Charcoal `#1F2937`
- Muted: Slate `#6B7280`

### Typography
- Headings: Sora
- Body: Source Sans 3
- Numerals: Tabular when presenting financial data

### UI Tokens (HSL for Tailwind/shadcn)
- --background: 45 33% 96%
- --foreground: 222 47% 11%
- --primary: 210 63% 23%
- --primary-foreground: 0 0% 100%
- --secondary: 178 47% 33%
- --secondary-foreground: 0 0% 100%
- --accent: 35 68% 68%
- --accent-foreground: 222 47% 11%
- --muted: 220 8% 46%
- --border: 220 13% 91%
- --ring: 210 63% 23%

### Spacing and Layout
- Grid: 12-column with 24px gutters
- Base spacing: 4px scale (4, 8, 12, 16, 24, 32, 48, 64)
- Radius: 10px default, 16px for cards

### Iconography and Illustration
- Line icons, consistent stroke weight
- Avoid cartoonish elements

## Content Guidelines
- Always show context on data changes
- Explain AI outputs with source and confidence
- Provide clear next actions

## Accessibility
- Minimum contrast ratio 4.5:1
- All interactive elements must have focus states
